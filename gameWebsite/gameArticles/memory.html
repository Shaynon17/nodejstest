How to Play Memory

Memory is a simple matching game for children.

Memory Rules

| Players: 2+ | Type: Matching | Supplies: 1 deck |

Objective

To match a pair by flipping two cards of the same rank.

Setup

Place all of the cards in the deck face down in rows and columns.

Game Play

The first player will flip over any two cards. If the cards are not the same rank, they are flipped back upside down and it is the next players turn. If the cards flipped are the same rank, the pair is kept by the player, and the player gets to go again. Players continue to take turns until all cards have been matched.

Winner

After all pairs have been matched, the player with the most pairs wins.

Variants

For a slightly more challenging version, make it to where the pairs have to be the same rank and color. Decide which version you are going to play before the game begins.
